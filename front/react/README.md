# TuniERP — Plateforme ERP Modulaire

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

TuniERP est une plateforme ERP modulaire conçue pour les entreprises tunisiennes. Solution complète de gestion : facturation, ventes, stock, CRM, comptabilité, RH et bien plus encore.

## Démarrage rapide

1. Cloner le projet

```bash
git clone https://github.com/talelbrini3/tunierpv2.git
```

2. Installer les dépendances

```bash
cd front/react
npm install
```

3. Lancer le serveur de développement

```bash
npm run dev
```

## Stack technique

- **Framework** : Next.js 16 + React 19
- **UI** : Material UI v7
- **Animation** : Framer Motion
- **Base de données** : Prisma 7 + PostgreSQL
- **Langage** : JavaScript / TypeScript
